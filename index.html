LRU Cache Visualizer

An interactive visualization of the Least Recently Used (LRU) cache algorithm. The LRU cache evicts the least recently used item when the cache is full and a new item needs to be added.

Cache Configuration

How LRU Works

  • When accessing an item, it becomes "most recently used"
  • When cache is full, the "least recently used" item is evicted
  • Cache hits are faster than cache misses
  • Try adding more items than cache size to see eviction
